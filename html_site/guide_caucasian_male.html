Caucasian male (M_CA01.160)

The male model M_CA01.160 has requested 160 iterations, in order to test the pectoral area and improve the topology and the weighting of the skeleton. As the female model, it provides an optimized mesh structure, morphable in any character and expression and easy to sculpt.

The image shows an example of character created using M_CA01.160

It's an average Caucasian man, with idealistic proportions, approximatively 30-33 years old.
