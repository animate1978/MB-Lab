Elf female (F_FT01.32)

F_FT01.32 is a template for female elf.

The topology is derived from FCA01 so it includes all the main muscles and tendons, with an advanced structure for the face expressions. The model provides teeth and detailed eyes with separated cornea and eyelashes.
